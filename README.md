# 💁‍♀️ 서비스 소개

   ### 사용자 감성 기반 책 추천 서비스, 북스비
   - 사용자의 성향과 책 선호도에 따라 감성 점수를 계산하여, 감성 점수와 일치하는 책 추천 서비스
   - 감정점수란?
        - 사용자가 선호한 책  + 해시태그 + 찜한 책
        - 기준의 description 단어들의 점수를  기반으로 7가지 색 중 유저의 색을 정해줌
